## Pip3 Install Ansible


### Precondition

Please using root user install python3/pip3 environment first.


### Modify ansible.cfg

```shell
host_key_checking = False
scp_if_ssh = True
```

### Manual Install

- sshpass (yum)
- parakimo (pip)
- argcomplete (pip)
